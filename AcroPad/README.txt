--  概要  --------------------------------------------------
  簡単操作ですぐに使えるシンプルなテキストエディターです。
  テキストやプログラムの編集にご利用ください。

--  動作環境  ----------------------------------------------
  アプリケーションの実行には、以下の環境が必要になります。
    ・Microsoft Windows 7 / 8 / 8.1 / 10
    ・Microsoft .NET Framework 4.6
